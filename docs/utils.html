Hiding elements

You can choose to hide on element based on the screen size:

  • All screen sizes .hidden
  • Mobile (less than 768px) .hidden-mobile
  • Tablet (more than 768px) .hidden-tablet
  • Laptop (more than 1024px) .hidden-laptop
  • Desktop (more than 1280px) .hidden-desktop

However, if you want to hide the element from sight but let it remain in the same place for screen readers, then you should use hidden-sr instead:

  • .hidden-sr
  • .hidden-sr-mobile
  • .hidden-sr-tablet
  • .hidden-sr-laptop
  • .hidden-sr-desktop
